# NetflixGPT
A GPT based chatbot for Netflix
## Features
- Sign in / Sign up
    - form
    - redirect to Browse
- Browse
    - Header
    - Main Section
        - Trailer in background
        - Title & Description
    - Movie Suggestions
        - Horizontal Scroll

## STEPS
- Create a React App / Parcel App
- Set up tailwindcss
- Set up react-router
- Create Header
- Login Form
- Sign up Form
- Form Validation
- Set up Firebase
- Project deployment on Firebase
- Sign up / Sign in with Firebase
- Redux setup
